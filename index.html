HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • src references the source of an image, alt describes the image.
  • Meta contains info not seen by person viewing the website.
  • Should only use h1 once(then proceed with h2-h6 for headers according to size).
  • use ul for unordered lists and li to list things
  • every HTML element is essentially a box

CSS

  • three ways to style, inline CSS, internal CSS style sheet, and external CSS style sheet
  • style an element directly in the HTML file, use inline CSS
  • CSS inline useful for quick and permanent changes
  • most common way to style a webpage is by using an external CSS style sheet
  • a selector defines the element or attributes to which the rules, or declarations, will apply
  • Declarations two important components:The property we want to apply and the value of the property
  • padding, located inside the element, margin is the transparent buffer outside an element
  • * is used to apply rule to all elements
  • Do Not Repeat Yourself (DRY) code.
  • Class attributes allow us to share a CSS rule to any element we choose by assigning the rule to a class attribute with a class selector

  • git pull origin main
  • git push (name of branch)
  • git -A ; preps stage
  • git commit -m (name of branch"

JavaScript

  • programming language that allows users to interact with websites
  • can power your websites with effects, interactivity, data communication, and much more.
  • Variables are containers that store values(After assigning a value to a variable, you can change it later in the code)
  • A semicolon at the end of a line indicates where a statement ends
  • JavaScript is case sensitive
  • Data Types: String, Number, Boolean, Array, Object
  • Comments are snippets of text that can be added along with code. The browser ignores text marked as comments( can use // to imply comments)
  • Operators: Addition, Subtraction, Division, Assignment, Strict Equality, Not, Does-not-equal
  • group of related data is called an array
  • strings need to be contained within quotation marks
  • A for loop uses the predictable pattern of indices to perform a task on all the items in an array
  • A for loop contains the following three important statements:first statement determines the starting point for our loop, second statement is the condition, final statement is what allows the array to iterate over each item
